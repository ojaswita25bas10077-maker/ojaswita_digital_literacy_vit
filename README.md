Project Overview
This project focuses on developing digital literacy skills, building a professional online presence, and understanding safe and responsible use of digital platforms. It includes practical tasks such as creating a digital portfolio, practising coding, using collaboration tools, learning email etiquette, and spreading awareness about cybercrime.


Module Summaries

Module 1: Digital Literacy Awareness Infographic
Designed an infographic using Canva covering topics like digital literacy, useful digital tools, safe internet practices, and professional online presence. This helped in understanding how to present information visually and effectively.

Module 2:Student Digital Portfolio
Created profiles on GitHub, LinkedIn, and Kaggle. Built a GitHub README and updated LinkedIn education details to establish a professional online presence.

Module 3: Coding & Collaboration Platforms
Practised coding on HackerRank by solving a beginner-level problem. Created a Digital Literacy Awareness Quiz using Google Forms and linked responses to Google Sheets.

Module 4: Email Etiquette & Social Media Awareness
Drafted professional emails for academic and internship purposes. Also created a checklist highlighting responsible social media usage.

Module 5: Cybercrime Awareness
Studied UPI fraud as a cybercrime case and created a prevention checklist to promote safe online practices among students.
